SenseReader COM Synth for NVDA

v 0.1.1

* 사용법

1. addon을 설치합니다.
2. 센스리더를 실행합니다.
3. NVDA를 실행하고 음성엔진을 SenseReader Synth로 변경합니다.
4. 센스리더 단축키 알트+쉬프트 + 백슬래시를 눌러서 센스리더를 일시중지시킵니다.
5. NVDA를 즐깁니다.

* 달라진 점

v0.1.1
xvmgr.py 전역 플러그인 추가
이 플러그인은 SenseReader Synth가 기본 설정이 된 채로 NVDA가 시작되는 경우 음성 엔진을 자동으로 espeak로 전환해 줍니다. 
센스엔진을 곧장 사용할 수는 없지만 음성 출력이 중단되지는 않도록 임시 처리를 한 것입니다.

v01.0
센스리더 엔진을 NVDA에서 사용하도록 함.
버그: NVDA를 재시작할 때 SenseReader Synth가 기본 설정된 경우 음성 출력이  안됨.
