SenseReader COM Synth for NVDA

버전 0.3.3

* 사용법

1. addon을 설치합니다.
2. 센스리더를 실행합니다.
3. NVDA를 실행하고 음성엔진을 SenseReader Synth로 변경합니다.
4. 센스리더 단축키 알트+쉬프트 + 백슬래시를 눌러서 센스리더를 일시중지시킵니다.
5. NVDA를 즐깁니다.

* 달라진 점

v 0.3.1
SenseReader Synth를 설정한 상태에서 NVDa를 재시작할 때 
센스리더가 실행되어 있다면 센스리더 엔진을 사용하고
센스리더가 실행되어 있지 않다면 espeak로 시작한다.
이 문제를 해결하려고 다시 xvMgr.py라는 전역 플러그인을 재작성함.
파라메터와 퍼센트를 전환하는 내부 메서드를 사용하여 설정값을 좀 더 자연스럽게 조정
모든 설정값은 센스리더와 같이 10단계입니다.



v 0.3.0
센스리더 Synth로 설정하고 NVDA를 재시작할 때 음성 출력이 되지 않던 문제를 일부 해결
COM 엔진을 제어하는 클래스를 cython으로 컴파일... 약간의 속도향상을 기대합니다.
불필요해진 전역 플러그인을 제거

v 0.2.0

음성 볼륨, 높이, 속도를 조절할 수 있습니다.

v0.1.1
xvmgr.py 전역 플러그인 추가
이 플러그인은 SenseReader Synth가 기본 설정이 된 채로 NVDA가 시작되는 경우 음성 엔진을 자동으로 espeak로 전환해 줍니다. 
센스엔진을 곧장 사용할 수는 없지만 음성 출력이 중단되지는 않도록 임시 처리를 한 것입니다.
